
# okhttp3-fast-spring-boot-starter
在`spring-boot`项目中方便的使用`okhttp`，快捷的创建`OkHttpClient`，并提供了简单易用的`POST`，`GET`，`PUT`，`DELETE`方法帮助你连接`HTTP`请求，本项目依赖`OkHttp`的版本为`3.14.9`。
